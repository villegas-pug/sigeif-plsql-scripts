# Reglas Preguntas:


## Campo: `AP_CONDICION`

- {"id":3525,"valor":"BECA 18"} ‚Üí  Se muestra, si la pregunta 3525(select) es seleccionada con "BECA 18"
- {"valor":"BECA 18"} ‚Üí  Se muestra, si la pregunta tiene un valore

## Campo: `AP_TIPO_CONTROL`

- cabecera | label | radio | select | selectM | text
  
## Campo: `AP_PREGUNTA2` & `AP_TIPO_CONTROL2`

> Es una subpregunta dentro de la pregunta, sirve para especificar un detalle.
